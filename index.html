☁️ Connect Firebase (Free)
Setup (free, ~3 minutes):
1. Go to console.firebase.google.com → Add project → any name
2. Build → Realtime Database → Create database → Start in test mode
3. Copy your database URL: https://YOUR-PROJECT-default-rtdb.firebaseio.com
4. ⚠️ Fix the Rules (required!): In Firebase → Realtime Database → Rules tab → replace with:
{ "rules": { ".read": true, ".write": true } } Click Publish. Then paste your URL below and connect.
✅ Firebase test mode is open for 30 days. After that, add a simple rule to keep it open, or just reset it — your data stays.
